# GitHub Repository Analyzer

> An AI-powered tool to analyze GitHub repositories using the GitHub API and Gemini AI.

## Overview

GitHub Repository Analyzer is a web application that leverages AI to provide deep insights into any GitHub repository. It uses the GitHub API for data access and Gemini AI for advanced analysis, code understanding, and summarization.

## Features

- **Repository Analysis:** Get high-level summaries and insights about any public or private GitHub repository.
- **Commit Analysis:** Understand the impact and content of individual commits, including code changes and summaries.
- **File Analysis:** Analyze specific files for complexity, purpose, and code quality.
- **Pull Request Analysis:** Summarize and review pull requests, including code diffs and discussion context.
- **Interactive Chat:** Ask questions about the repository and receive AI-generated answers.
- **Streaming Analysis:** Real-time streaming of analysis results for large files or repositories.

## Usage

1. **Connect a Repository:** Provide a GitHub repository URL and a Personal Access Token (with appropriate permissions) to connect.
2. **Explore:** Use the dashboard to view repository stats, recent commits, files, and pull requests.
3. **Analyze:** Select any commit, file, or pull request to receive AI-powered analysis and summaries.
